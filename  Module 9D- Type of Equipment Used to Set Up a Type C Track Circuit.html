Module 9D: Type of Equipment Used to Set Up a Type C Track Circuit

Setting up a Type C track circuit necessitates the use of specialized equipment designed to generate, transmit, and analyze two distinct AC frequencies simultaneously:

  • Dual-Frequency Transmitter: Generates both the train detection and broken rail detection frequencies with stable and controlled output levels.
  • Dual-Frequency Receiver: Separates and analyzes each frequency using filters and detection circuits to monitor changes in amplitude and/or phase.
  • Insulated Rail Joints: Installed on both rails to provide electrical isolation for the signaling currents.
  • Connecting Cables: Designed with impedance-matching characteristics to minimize loss and distortion of signals.
  • Spectrum Analyzer: Visualizes the frequency spectrum, ensuring both signals are clean, stable, and at the correct levels during setup and troubleshooting.
  • Oscilloscope: Examines waveform shape, phase relationships, and potential signal distortion or interference.
  • Multimeter (AC Measurement): Measures voltage and current of each signal for basic circuit checks and calibration.
  • Shunt Application Device: Simulates train presence for testing the response of the detection circuitry.
