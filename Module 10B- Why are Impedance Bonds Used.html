🚦 Module 10B - Why are Impedance Bonds Used?

Purpose of Impedance Bonds

Impedance bonds are important components in railway signaling systems, especially in electrified territory where the running rails are used as part of the traction return current path.

In electrified railways, traction return current must travel back to the power substation. The running rails often provide this return path, so the current must be able to pass through track circuit areas safely and continuously.

Track circuits are also connected to the rails for train detection. This means the rails may carry both traction current and signaling current at the same time.

Impedance bonds allow these two systems to share the rails without interfering with each other. They allow traction return current to pass while helping keep signaling current within the correct track circuit.

One major reason impedance bonds are used is to provide a path around insulated rail joints. Insulated joints separate track circuits, but traction return current still needs a continuous path through the rails.

Without impedance bonds, traction return current could be interrupted at insulated joints. This could reduce traction power efficiency, create unsafe current paths, or cause interference with signal equipment.

Impedance bonds also help prevent traction current from interfering with track circuit operation. They present low impedance to traction current and higher impedance to signaling frequencies.

This allows adjacent track circuits to remain electrically isolated for signaling purposes while still allowing traction return current to flow through the railroad system.

In simple terms, impedance bonds make it possible for high-power traction systems and low-power signaling systems to safely use the same rails.
